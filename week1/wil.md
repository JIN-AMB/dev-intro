개발입문스터디 1주차

git: 버전관리시스템
github: 코드 등록 및 저장소 호스팅 클라우드 서비스
vercel: 배포

git명령어들
git status: 어떤 파일이 수정되었는지, 어떤 파일이 새로 생겼는지, 혹은 add가 되었는지를 알려주는 역할
git add <file>: 저장하고 싶은 파일을 골라서 스테이징영역에 올리는 것
git commit -m"메세지": add로 담아둔 파일을 묶어 하나의 '버전'으로 만들고 -m뒤의 메세지는 이 버전에서 무엇을 바꿨는지 적는 메모
git push: git hub에 백업하고 공유
